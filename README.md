# AdventureWorks Sales Dashboard (Power BI)

## Overview
An interactive Power BI dashboard built using the AdventureWorks dataset to analyze sales performance, customers, products, and geographic trends.

## Features

- Executive Dashboard
- Sales KPI Tracking
- Customer Analysis
- Product Analysis
- Geographic Analysis
- Year-based filtering
- Dynamic DAX measures
- Interactive slicers

## Tools Used

- Power BI Desktop
- DAX
- Power Query
- Data Modeling

## Key Metrics

- Total Sales
- Previous Year Sales
- Year-over-Year (YoY) Growth
- Product Ranking
- Running Sales

## Skills Demonstrated

- Data Modeling
- Star Schema
- DAX Calculations
- Dashboard Design
- Business Intelligence
- Data Visualization
